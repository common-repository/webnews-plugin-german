== Description ==

Webnews Plugin (german)

Es wurde lange Zeit versprochen. Nun ist es endlich fertig! Das Plugin wird hier vorab als Beta-Version veröffentlicht.
Wir sind gespannt auf dein Feedback. Webnews unterstützt Wordpress ab Version 2.8.x. Wir wissen daher nicht, ob es auch mit älteren Versionen von Wordpress funktioniert.

== Installation ==

I) Installation - Fertig in 5 Minuten

1) Melde dich bei deinem Blog als Admin an. http://deineDomain/wp-admin
2) Unter 'Plugins' > 'Installieren', klicke auf 'Hochladen'. Es öffnet sich ein Dialog. Wähle das Plugin 'webnewsde.zip' von der Stelle aus, wo du sie in Schritt 1) abgespeichert hast. Bestätige den Dialog mit 'Ok'.
3) Klicke nun auf "Jetzt installieren". Fertig.
 
II) Konfiguration

1) Gehe nun zu 'Einstellungen' > 'Webnews.de'.
2) Setze ein Häkchen bei "Webnews Plugin aktivieren"
3) Gib nun darunter deine Webnews Nutzerdaten ein. Wenn du noch keine Profil bei Webnews hast, kannst du hier eins erstellen.
4) Bestätige die Einstellungen mit Änderungen speichern".
 
III) Schreibe deine erste Nachricht bei Webnews direkt aus Wordpress

1) Erstelle wie gewohnt einen neuen Artikel.
2) Du benötigst zumindest eine Überschrift, Artikeltext und ein Stichwort.
3) Unter dem Eingabefeld gibt es einen neuen Abschnitt "Webnews.de"
4) Wähle die Webnews Kategorie aus, in der du deinen Beitrag veröffentlichen willst.
5) Setze ein Häkchen bei "auf Webnews veröffentlichen", und klicke den Wordpress veröffentlichen Button.
6) Wenn du einen Artikel bearbeiten möchtest, und er ist bereits bei Webnews verlinkt, wird dir der entsprechende Link angezeigt.

Allen Bloggern viel Spaß beim ausprobieren. 

Hinweis: Die Verwendung des Plugins geschieht auf eigene Gefahr. Webnews haftet nicht für etwaige Schäden an Script oder Daten.
